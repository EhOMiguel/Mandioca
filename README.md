# mandioca

O projeto **mandioca** é uma API desenvolvida com Java Spring para assinatura de documentos digitais. Este repositório serve como um ambiente de testes e aprendizado para explorar funcionalidades relacionadas à assinatura digital.

## Funcionalidades

- **Assinatura de Documentos Digitais**: Assine documentos digitais e eficiente.

## Tecnologias Utilizadas

- **Linguagem**: Java
- **Framework**: Spring Boot
